README

**Установка**
1. Создать пользователя server-admin
2. Создать виртуалхост server-admin.bpt.loc
3. В public_html скопировать папку ban-service
4. В файл /etc/rc.local прописать запуск демона: php /home/server-admin/server-admin.bpt.local/public_html/ban-service/daemon/daemon.php

**Система**
Debian 8 Jessie x64
Apache 2.4.10
Iptables 1.4.21

**Изменения**
Альфа версия программного продукта
Система предоставляет web-интерфейс для iptables, позволяющий управлять форвардингом пакетов по ip адресам.

Для предоставления форвардинга узлу локальной сети необходимо
-> ввести его ip - адрес
-> нажать кнопку "Разрешить"
-> нажать кнопку "Сохранить изменения"

Для запрета форвардинга узлу локальной сети необходимо
-> нажать кнопку "Х" рядом с ip - адресом, соответствующим нужному узлу
-> нажать кнопку "Сохранить изменения"
